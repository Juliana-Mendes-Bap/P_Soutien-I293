# P_Soutien-I293
Website du P_Soutien du module I293
Dans ce P_Soutien on avait des exercices a faire sur un site, et peu a peu les exercices devennaient un website qui explique quelque chose (fraises, legumes, fruits, animaux, etc...)
Le Website a un responsive pour la taille d'un Iphone 16 PRO MAX, et pour les ordis.
